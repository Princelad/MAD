# Practical 6 - Gujarat Government Schemes App

A minimal Flutter application showcasing various Gujarat Government schemes and welfare programs.

## Features

- **Clean Dark UI**: Uses Catppuccin-inspired color scheme consistent with other practicals
- **Government Schemes Display**: Lists popular government schemes with categories
- **Interactive Details**: Tap on any scheme to view detailed information
- **Categorized Schemes**: Organized by Agriculture, Health, Employment, etc.
- **Responsive Design**: Works across different screen sizes

## Included Schemes

1. **PM-KISAN** - Financial assistance to farmers
2. **Beti Bachao Beti Padhao** - Girl child education and empowerment
3. **Ayushman Bharat** - Health insurance for poor families
4. **MGNREGA** - Rural employment guarantee scheme
5. **Jan Aushadhi** - Affordable medicines for all
6. **Digital Gujarat** - Digital services and governance

## Technical Details

- **Framework**: Flutter
- **Architecture**: Single page with modal dialogs
- **Widgets**: ListView, Cards, AlertDialog
- **State Management**: StatelessWidget (minimal app)
- **Color Scheme**: Dark theme with blue accents
